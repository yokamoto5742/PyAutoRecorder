# 変更履歴

このプロジェクトのすべての重要な変更は、このファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に基づいており、
バージョン番号は [Semantic Versioning](https://semver.org/lang/ja/) に従っています。

## [Unreleased]

## [1.1.0] - 2026-07-16

### 追加
- 手順書プレビューペインをメインウィンドウに追加
- マニュアルプレビューの表示・サイズをconfig.iniで設定可能に
- 選択行直下への項目挿入機能を追加
- 制御対象コントロール（マウス・キーボード・ウィンドウ選択）の有効/無効を個別に切り替える機能を追加

### 修正
- COMError発生時のウィンドウ消滅に対応し、エラーハンドリングを強化

## [1.0.1] - 2026-07-13

### 追加
- カーソル下の要素をハイライトする機能を追加（app/element_highlight）
- マクロ・ワークフロー再生時にクリップボード変数の確認と入力に対応
- 変数入力用のキー '{VAR:}' をサポート

## [1.0.0] - 2026-07-08
- 初版リリース
